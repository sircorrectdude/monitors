MONITORS
--------------------------------------------------------------------------------

To get started, please complete the following steps:

I) Configure the Server Applications

1. Download and install a MySQL 5.x database from 
   http://dev.mysql.com/downloads/mysql/5.0.html#downloads.

2. Run "mvn jetty:run-war" and view the application at http://localhost:8080.

3. More information can be found at:

II) Configure the Client

Best Practice is to use FireFox on the Client Machine

1. With Firefox on the Client Machine goto:
	https://addons.mozilla.org/en-US/firefox/addon/full-fullscreen/
	and install the  AddOn Full Fullscreen 
	By press F11 you can toggle Fullscreen Mode
2. Enter ServerAdress as Himepage in Firefox	
3. Start Firefox on startup (f.r Ubuntu: System --> Preferences --> Startup Applications command: firefox %u)
